# 07 - Portfolio Projects

## Required Projects

### 1. ML Classification Model
- Predict outcomes from tabular data
- Folder: `ml_projects/`
- Skills: Scikit-learn, feature engineering

### 2. Image Classifier (CNN)
- PyTorch or TensorFlow
- Folder: `dl_projects/`
- Skills: CNNs, transfer learning

### 3. RAG Chatbot
- Q&A over custom documents
- Folder: `llm_projects/`
- Skills: LangChain, vector DBs, embeddings

### 4. Multi-Agent System
- Autonomous task completion
- Folder: `agent_projects/`
- Skills: CrewAI, orchestration

## Each Project Should Have
- README with problem description
- Clean, documented code
- Results/metrics
- Demo or screenshots
