# GEO Quick Reference Card

## The 40-60 Rule
- **AnswerBox**: 40-60 words
- **Direct answer first**, then supporting details
- Include unique value proposition

## Q&A Formula
1. **Is [tool] really free?** → Yes + privacy angle
2. **What formats supported?** → List + requirements
3. **How long does it take?** → Time + speed advantage
4. **Is it safe?** → Privacy + security features
5. **Mobile support?** → Any device + optimization
6. **vs Competitor?** → 3 unique advantages
7. **Why choose us?** → No install + privacy + free

## Schema Must-Haves
- Article (basic info)
- FAQPage (5-7 Q&As)
- HowTo (3-5 steps)
- BreadcrumbList (hierarchy)

## Internal Linking
- 2-3 contextual links per page
- Related tools section
- Use natural anchor text
- Always add UTM parameters

## Update Schedule
- **Daily**: Price monitoring (automated)
- **Weekly**: Performance review
- **Monthly**: Content refresh
- **Quarterly**: Complete overhaul

## Success Metrics
- **Citation Rate**: Target 15-25%
- **AI Traffic**: Target 200K/month
- **Conversion**: Target 20-25%
- **Query Coverage**: Target 40-60%

## Platform Optimization
- **ChatGPT**: Encyclopedic tone, numbered lists
- **Perplexity**: Fresh content, Reddit examples
- **Gemini**: Schema markup, multilingual
- **Claude**: Scholarly depth, citations

## Emergency Actions
- Traffic drops >20%: Check for technical issues
- Citation drops >30%: Refresh content immediately
- New competitor: Update comparison table
- Algorithm change: Review all content

## Quick Wins
1. Add "Updated [date]" to all content
2. Include statistics every 150-200 words
3. Write H2s as questions
4. Add TL;DR boxes
5. Use tables and lists

## Common Mistakes
- ❌ Too long AnswerBox
- ❌ Forgetting UTM parameters
- ❌ Not updating content
- ❌ Ignoring mobile experience
- ❌ No E-E-A-T signals

## Remember
