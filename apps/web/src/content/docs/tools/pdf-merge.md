# PDF Merge

Combine multiple PDF files into one document, entirely in your browser.

## Overview

The PDF Merge tool lets you combine PDFs in any order. Perfect for:

- Merging scanned documents
- Combining report sections
- Creating portfolios
- Assembling tax documents

## How to Use

### 1. Upload PDFs

Drag and drop your PDF files into the drop zone, or click to browse.

### 2. Arrange Order

Drag files to reorder them. The top file will be first in the merged document.

### 3. Merge and Download

Click **Merge PDFs** and download your combined document.

## Features

| Feature | Description |
|---------|-------------|
| **Batch Upload** | Select multiple files at once |
| **Drag to Reorder** | Visual ordering interface |
| **Preview** | See first page of each PDF |
| **No Size Limit** | Browser memory is the only limit |
| **Fast Processing** | Merge in seconds |

## Tips

### File Order
Files are merged in the order shown. Drag to rearrange before merging.

### Large Files
For best performance with large PDFs:
- Close other browser tabs
- Use Chrome or Edge for better memory handling
- Process in batches if needed

### Page Orientation
The tool preserves original page orientations and sizes.

## Troubleshooting

**"File too large" error**: Your PDF may be corrupted or extremely large. Try compressing it first with our [PDF Compressor](/tools/pdf-compressor).

**Merged file won't open**: The source PDF may be password-protected or corrupted.
