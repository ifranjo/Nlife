# Quick Start Guide

Get started with New Life Solutions tools in three simple steps.

## Step 1: Choose a Tool

Browse our [tools directory](/tools) or use the search bar to find what you need. Popular starting points:

- [PDF Merge](/tools/pdf-merge) - Combine multiple PDFs
- [Image Converter](/tools/image-converter) - Convert between image formats
- [QR Generator](/tools/qr-generator) - Create QR codes instantly

## Step 2: Upload Your Files

Each tool has a **drop zone** where you can:

1. **Drag and drop** files from your computer
2. **Click to browse** and select files
3. **Paste** images directly (on supported tools)

### Supported File Types

| Tool Type | Supported Formats | Max Size |
|-----------|------------------|----------|
| PDF Tools | PDF | 50 MB |
| Image Tools | JPG, PNG, WebP, GIF, BMP, TIFF | 10 MB |
| Video Tools | MP4, WebM, MOV, AVI | 500 MB |
| Audio Tools | MP3, WAV, OGG, FLAC | 100 MB |

## Step 3: Process & Download

1. Configure any options (quality, format, etc.)
2. Click the **Process** or **Convert** button
3. Wait for processing to complete
4. Download your result

## Example: Converting an Image

Let's convert a PNG to JPG:

1. Go to [Image Converter](/tools/image-converter)
2. Drag your PNG file into the drop zone
3. Select "JPG" as the output format
4. Adjust quality if desired (default is 85%)
5. Click **Convert**
6. Download the JPG file

That's it! The entire process takes seconds.

## Tips for Best Results

### File Size
- Smaller files process faster
- For images, consider compressing before processing
- Videos over 100MB may take longer

### Batch Processing
Many tools support multiple files at once:
- Select multiple files in the file picker
- Or drag multiple files to the drop zone
- Process them all together

### Quality Settings
- Higher quality = larger file size
- 80-90% is usually the sweet spot for images
- PDF compression has "Low/Medium/High" presets

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl/Cmd + O` | Open file picker |
| `Ctrl/Cmd + S` | Download result |
| `Esc` | Close modals |
