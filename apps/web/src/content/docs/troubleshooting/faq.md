# Frequently Asked Questions

## General Questions

### Are these tools really free?

**Yes!** All tools are completely free to use. No subscriptions, no watermarks, no hidden fees.

### Do I need to create an account?

**No.** Our tools work without any account or login. Just visit the page and start using them.

### Is there a file size limit?

File size limits vary by tool:

| Tool Type | Max File Size |
|-----------|---------------|
| PDF Tools | 50 MB |
| Image Tools | 10 MB |
| Video Tools | 500 MB |
| Audio Tools | 100 MB |

### Can I use these tools commercially?

**Yes!** Processed files are yours to use however you like, including commercial use.

## Privacy & Security

### Do you store my files?

**No.** All processing happens in your browser. Files never leave your device or touch our servers.

### Is it safe to process sensitive documents?

**Yes.** Since nothing leaves your device, it's actually safer than cloud-based alternatives for sensitive data.

### Do you track what I'm doing?

We collect anonymous usage statistics (which tools are popular) but never track file contents or personal information.

## Technical Issues

### The tool isn't working. What should I do?

1. **Refresh the page** - Sometimes a simple reload fixes issues
2. **Try a different browser** - Chrome and Edge have the best compatibility
3. **Check file format** - Ensure your file is in a supported format
4. **Disable extensions** - Ad blockers can sometimes interfere

### Why is processing slow?

Large files naturally take longer. Tips for faster processing:
- Close other browser tabs
- Use a modern browser (Chrome, Edge, Firefox)
- Ensure you have available RAM
- For videos, shorter clips process faster

### Can I use these tools offline?

**Yes!** After the first visit, most tools work offline thanks to our Progressive Web App (PWA) support.

## File Formats

### What image formats are supported?

**Input:** JPG, PNG, WebP, GIF, BMP, TIFF, SVG
**Output:** JPG, PNG, WebP, GIF

### What video formats work?

**Input:** MP4, WebM, MOV, AVI, MKV, FLV
**Output:** MP4, WebM, GIF

### Can I convert scanned PDFs to text?

Yes! Use our [OCR Tool](/tools/ocr) to extract text from scanned PDFs and images.
