# Thumbnail Prompts for AI Image Generation

Style guide for generating consistent tool thumbnails.

## Base Style

```
Style: Dark, enigmatic, minimal, monospace aesthetic
Background: Pure black (#0a0a0a) with subtle grid pattern
Lighting: Soft cyan/white glow, low-key lighting
Mood: Mysterious, futuristic, clean
Format: 1200x630px (OG image), 400x300px (card thumbnail)
```

## Tool Thumbnails

### PDF Merge
```
Prompt: Minimalist dark composition. Two floating translucent PDF document icons
merging into one, connected by thin white geometric lines. Subtle cyan glow at
connection point. Grid pattern background. Pure black (#0a0a0a). Clean, enigmatic,
futuristic. No text.

Negative: Colorful, busy, realistic paper, bright colors, gradients
```

### PDF Split
```
Prompt: Minimalist dark composition. Single translucent PDF document icon splitting
into multiple smaller fragments. Thin white scissor-like geometric lines cutting
through. Particles dispersing. Grid pattern background. Pure black (#0a0a0a).
Clean, enigmatic, futuristic. No text.

Negative: Colorful, busy, realistic scissors, bright colors
```

### Image Compress
```
Prompt: Minimalist dark composition. Large translucent image frame compressing into
smaller version. Geometric compression arrows. Data visualization particles.
Grid pattern background. Pure black (#0a0a0a). Subtle green efficiency glow.
Clean, enigmatic, futuristic. No text.

Negative: Colorful, realistic photos, bright colors, busy
```

### QR Generator
```
Prompt: Minimalist dark composition. Floating geometric QR code pattern forming from
scattered white pixels. Subtle grid alignment guides. Pure black (#0a0a0a) background.
Soft white glow emanating from QR pattern. Clean, enigmatic, futuristic. No text.

Negative: Colorful, busy, realistic phone, bright colors
```

### AI Translator (Pro)
```
Prompt: Minimalist dark composition. Two floating text blocks in different scripts
connected by thin translucent neural network lines. Soft blue AI glow at center.
Grid pattern background. Pure black (#0a0a0a). Clean, enigmatic, futuristic. No text.
Subtle "locked" feel with thin border.

Negative: Colorful flags, busy, bright colors, realistic
```

### Video Avatar (Pro)
```
Prompt: Minimalist dark composition. Abstract human silhouette face dissolving into
geometric particles. Holographic wireframe effect. Soft purple AI glow.
Grid pattern background. Pure black (#0a0a0a). Clean, enigmatic, futuristic. No text.
Subtle "locked" feel with thin border.

Negative: Realistic face, colorful, busy, bright colors
```
